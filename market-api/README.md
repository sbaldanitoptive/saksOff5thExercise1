# Market API

---

## Software Requirements

- NodeJS version 16.13.0
- mysql-server
- imagemagick

## Setup

- Using Docker-Compose
  `docker-compose up`

## Open App

- Open Home Page at http://localhost:8080/
- Or check API Docs at http://localhost:8080/api-docs/
